## Why

Командный отчёт показывает агрегированные цифры по действиям, но не видно вклад каждого игрока. Тренеру нужно понять, кто именно выполнял действие (например, пасы, отборы), не переключаясь на индивидуальные отчёты по каждому игроку.

Сейчас у строки действия есть только drill-down «Комментарии»; разбивки по игрокам нет.

## What Changes

- В **командном** отчёте у каждой строки действия — кнопка разворота «По игрокам» (рядом с «Комментарии»)
- По клику — подтаблица: игрок → те же метрики (всего / успех / неудача / %), что и в основной строке
- Отдельная строка для командных событий (`SubjectType='team'`) — по имени команды
- Работает в плоском отчёте и в режиме «по матчам»; учитывает фильтры даты и турнира
- Индивидуальный отчёт без изменений (субъект уже один игрок)
